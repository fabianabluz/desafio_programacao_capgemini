# Desafio de programação da Capgemini

Esse é um projeto Java com o objetivo de resolver o questionário proposto pela empresa Capgemini como desafio.

# Pré Requisitos

Para que seja possível rodar essa aplicação é necessário atender alguns requisitos básicos.

 - java 8+
 - JUnit 4

# Como rodar a aplicação

É importante ter o JDK instalado em seu computador e as variáveis de ambiente configuradas devidamente.
